# Shared Colab Notebooks

Colab で使うノートブックを共有するためのリポジトリです。

[![Open In Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/spgti/shared-notebooks/blob/main/%E3%83%8E%E3%82%A4%E3%82%BA%E5%87%A6%E7%90%86%E3%83%84%E3%83%BC%E3%83%AB.ipynb)

## 公開中のノートブック

- [ノイズ処理ツール.ipynb](./ノイズ処理ツール.ipynb)

## 使い方

1. 上の `Open In Colab` ボタンを押します。
2. Colab 上でノートブックを開きます。
3. 必要に応じて `ランタイム` から GPU / CPU を切り替えて実行します。
